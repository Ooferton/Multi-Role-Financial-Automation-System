# 🧠 Sentience V3 | MARL Swarm Financial AI

Private, modular financial AI system powered by a **V3 Cyborg PPO Neural Network** with a **Multi-Agent Reinforcement Learning (MARL) Swarm** architecture.

## Architecture
- **V3 Brain**: PPO model with 23-dimensional observation space (price, RSI, MACD, OFI, NLP sentiment, market regime)
- **MARL Swarm**: 5 sector-specialized agents (TECH, FINANCE, HEALTH, ENERGY, CRYPTO)
- **Meta-Orchestrator**: Dynamic capital allocation based on sector Sharpe Ratios
- **LLM Supervisor**: Autonomous risk constraint rewriting via Gemini

## Features
- Live trading via Alpaca (paper + live modes)
- 59-ticker coverage across all sectors including Bitcoin/Crypto
- Adversarial crisis simulation during training
- Nightly AI journal & weekly SOUL.md rule updates
- Streamlit dashboard with real-time sector monitoring
